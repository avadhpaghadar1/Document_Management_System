# Formatting Guide (Word)

This project report is provided as Markdown so it’s easy to edit. For college submission, paste it into Microsoft Word (or Google Docs) and apply the required format.

## Required Format (As Provided)

- Paper size: A4
- Font style: Times New Roman
- Main Heading: 16 pt, Bold
- Subheading: 14 pt, Bold
- Content text: 12 pt
- Line spacing: 1.5
- Margins: Minimum 1.5 inch on all sides
- Page numbers: Bottom center

## Quick Steps (Microsoft Word)

1. Open Word → create a blank document.
2. Page Layout / Layout:
   - Size → A4
   - Margins → Custom Margins → set Top/Bottom/Left/Right = 1.5"
3. Select all text (Ctrl + A):
   - Font → Times New Roman
   - Font size → 12
   - Line spacing → 1.5
4. Apply heading styles:
   - Main headings (TITLE PAGE, ACKNOWLEDGEMENT, TABLE OF CONTENTS, COMPANY PROFILE, CHAPTER 1 – PROJECT PROFILE, ABOUT THE TOOL): 16 pt Bold
   - Subheadings (e.g., 1.1 Company Overview): 14 pt Bold
5. Insert page numbers:
   - Insert → Page Number → Bottom of Page → Center
6. Update placeholders:
   - Replace [Your Name], [College Name], [Company Name], etc.

## Tip

After pasting, review spacing and page breaks so Title Page and Acknowledgement start on new pages if required by your college.
